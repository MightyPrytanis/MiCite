Both Modes Are Designed For Confidential Legal Work

Web App

  • Static Vercel/GitHub deployment.
  • Core formatting review runs in the browser.
  • Optional parallel-citation lookup sends only extracted case-citation identifiers.
  • No account system or database.
  • No third-party scripts on the checker page.
  • Document text and surrounding legal context remain in browser memory unless the user copies or downloads them.

Local Install

  • Runs from local files after unzipping.
  • Works without a hosted processing service.
  • Uses bundled logo and icon assets.
  • No telemetry, model training, or required remote citation API.
  • No command line or installer required.
MiCite checks citation format only. It does not validate the existence of a cited case or source, verify quotations or legal inferences, or determine whether an authority has been overruled, superseded, or otherwise limited. MiCite is a formatting tool, not a cite-checking or legal-research tool.
